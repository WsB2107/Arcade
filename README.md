# Dungeon Crushers

## Пояснительная записка

Автор: Оганян Тигран, Ермиличева Алена



### Описание проекта:
Dungeon Crushers - это аркадный платформер в жанре слешер, в стиле дарк фентези. 
#### Чтобы запустить игру, выберете файл main
- прежде чем начать игру, надо зарегистрировать аккаунт
- после регистрации, вы попадаете в главное меню
- из главного меню можно выбрать вкладку регистрации другого аккаунта, таблицы рекордов, настроек и самой игры
- в настройках можно посмотреть управление и настроить звук в самой игре
- в таблице рекордов выберете уровень, и сравните свой результат с другими игроками

#### Суть игры - стать королем подземелья, что таит в себе гору сокровищ:
- поочередно проходите уровни
- каждый уровень будет сложнее предыдущего, а также со своей уникальной игровой механикой
- уровни соответствуют своему дизайну как визуальному, там и звуковому
- проберитесь сквозь гору монстров и препятствий, чтобы достичь своей цели
- уровни можно проходить на скорость, улучшая свои рекорды.
