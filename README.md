# Dungeon Crushers

## Пояснительная записка

Автор: Оганян Тигран, Ермиличева Алена



### Описание проекта:
Dungeon Crushers - это аркадный платформер в жанре слешер, в стиле фентези. 

#### Суть игры:
 - выбрав режим игры и персонажа, вы попадаете на уровень
 - по ходу уровня на вас будут нападать **враги**
 - **_отбивайтесь_** от них при помощи своих способностей и пробивайтесь дальше
 - **_проберитесь_** через толпу враuов как можно быстрее
 - в конце **_сразитесь_** с **боссом**.
 - после прохождения уровня, вы увидете насколько быстро прошли уровень

## Техническое задание
 - Создать класс пользователя
   - создание диалогового окна с выбором пользователя или регистрацией нового
 - Создать класс базы данных
   - подключение базы
   - получение пользователей из базы данных
   - сохранение и получение прогресса
   - добавление и получение рекорда
 - Создать класс главного меню
   - главное меню
   - меню выбора режима игры 
   - меню настроек 
   - таблица рекордов
   - меню выхода
 - Создать класс персонажей
   - отображение выбора персонажа
   - получение персонажа
   - характеристики персонажей
 - Создать класс уровней
   - отображение выбора уровня
   - получение уровня
   - характеристики уровня
 - Создать класс настроек игры
   - настройка звука
   - настройка управления 
   - отображение меню настроек 
   - загрузка настроек 
 - Создать класс рекордов
   - обновление рекордов
   - сохранение рекордов
